doctype

content description
< !DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Transitional//EN" "http://www.w3.org/TR/xhtml1/DTD/xhtml1-transitional.dtd"> 要求非常宽松的DTD，它允许你继续使用HTML4.01的标识,目前应用比较多的doctype
< !DOCTYPE html> html5时代的doctype,目前杭州站和城市站首页都应用了这种声明，为了使用html5标签
用时把< !中间空格去掉

CSS书写顺序

name content
显示属性 display | position | float | clear | cursor ...
盒模型 margin | padding | width | height
文本属性 vertical-align | white-space | text-decoration | text-align | color | font | content ...
边框背景 border | background
内容管理、层级 overflow | z-index | zoom

css缩写

name content
border {border:1px solid #333;}
background {background:#333 url() no-repeat 50% 50%;}
margin,padding {margin:5px 6px 7px 8px;padding:5px 6px 7px 8px}
如果四个值一样，更精简的{margin:5px;padding:5px;}
font {font:700 12px/18px 'microsoft yahei';}

hack写法 

#test{width:10px;}
#test{width:8px\9\0;} /* IE9 */
:root #test{width:8px;} /* IE9,OPERA */
:root #test{width:8px\0;} /* IE9 */
#test{width:8px\0;} /* IE8+ */ 
#test{width:8px\9;} /* IE6+ */
#test{*width:6px;}  /* IE6、7 */
#test{_width:5px;}  /* IE6 */


常用css3兼容性写法

name content
opacity(会改变内部元素透明度) {opacity:0.4;filter:alpha(opacity=40);}
opacity(不会改变内部元素透明度) {background:rgba(255,255,255,.8);filter: progid:DXImageTransform.Microsoft.gradient(enabled='true',startColorstr='#ccffffff',endColorstr='#ccffffff');}
gradient {background:#f1a400;background:-webkit-linear-gradient(top, #eaa400, #f1a400);background:-moz-linear-gradient(top, #eaa400, #f1a400);}
