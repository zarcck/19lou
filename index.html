AM 模块加载器

Usage

AM(arg, arg, ...);

注意：

1. AM 将自动加载 jquery.js 与 App.component.js

2. 智能判断文件依赖关系，同个 AM 函数中任一模块加载失败都会导致后续模块停止加载

Options

Name type description
arg string | function

arg 可以被定义为以下形式：

  • 文件名称

    AM("showWin", "App.showWin", "App.showWin.js");

  • 相对路径

    AM("lib/mustache.js", "$forum/lib/mustache.js");

  • 绝对路径

    AM("http://i3.sbsnew.com/trunk/js/forum/lib/jquery.js");

  • 匿名函数

    AM(function() {...});

提示：

1. 文件名称缺失前缀或后缀时，将智能补全 App. 前缀与 .js 后缀

2. 标识符 $ 指代 JS 根目录，严禁在此标识符后添加 /

3. 模块将被加载，函数将被执行

App.login 登录

Usage

AM("login", function() {
    App.login({
        isLogin: "登录状态",
        host: "当前域名",
        ...
    });
});

初始化登录组件后，为链接添加 class="J_login" 来实现冒泡事件绑定。

用户处于登录状态，点击链接时按以下优先级进行处理：

1. 跳转 href 地址

2. 执行 data-callback 函数，并将当前点击对象当作第一个参数传入

3. 刷新当前页面

<a class="J_login" href="跳转" data-title="标题" data-ttname="注册来源" data-callback="回调">登录</a>

Options

Name type default description
isLogin boolean false 登录状态
host string 'http://www.19lou.com' 当前域名
qqLogin boolean true

QQ 登录

qqLogin: true  开启

qqLogin: false 关闭
sinaLogin boolean true

新浪登录

sinaLogin: true  开启

sinaLogin: false 关闭
taobaoLogin boolean true

淘宝登录

taobaoLogin: true  开启

taobaoLogin: false 关闭
qihooLogin boolean true

360 登录

qihooLogin: true  开启

qihooLogin: false 关闭
guide number -1

登录引导（单位：秒）

guide: -1 不显示

guide: 3  3秒后显示

AJAX 异步请求

Usage

$.ajax({
    type: "请求方式",
    url:  "请求地址",
    data: "发送到服务器的数据",
    success: function(res) {
        // 请求成功后的回调函数，res为服务器返回数据
    }
    ...
});
以上是 $.ajax 的传统写法，1.5.0 版本后 $.ajax 返回的是 deferred 对象，可以做如下链式操作：
$.when(
    $.ajax({
        type: "请求方式",
        url:  "请求地址",
        data: "发送到服务器的数据"
        ...
    })
).done(function(res) {
    // 请求成功后的回调函数，res为服务器返回数据
});

Options

以下只列举 $.ajax 的常用配置项，完整文档请 点击这里 查看。
Name type default description
type string 'GET'

请求方式，'POST' 或 'GET'
url string 当前地址

请求地址
data object | string

发送到服务器的数据
success function | array

请求成功后的回调函数

可以接受一个函数数组，每个函数将被依次调用。
